# I cannot decrypt RSA

https://alpacahack.com/daily/challenges/i-cannot-decrypt-rsa

## 問題の概要

RSA で暗号化したフラグを間違った方法で復号しようとして、復元できなかったフラグが与えられています。

- $p$, $q$ ： 素数
- $n = p \cdot q$
- $e = 65537$
- $m$ ： フラグを整数に変換したもの
- $c = m^e \mod n$ ： 暗号文
- $\phi = (p+1)(q+1)$ ： 間違ったオイラー関数
- $d$ ： $e \cdot d \equiv 1 \pmod \phi$ を満たす整数
- $m' = c^d \mod n$ ： 間違った復号結果

与えられているのは $n$, $e$, $\phi$ と $m'$ を bytes に変換した `flag` です。

## 解法

今回は $p$ と $q$ が求まりそうなので計算してみます。

$n=p \cdot q$ と $\phi = (p+1)(q+1)$ から $p$ と $q$ の和と積

$$
\begin{align*}
p+q &= \phi - n - 1 \\
p \cdot q &= n \\
\end{align*}
$$

が分かります。したがって､ $p$ と $q$ は二次方程式

$$(x-p)(x-q) = x^2 - (\phi - n - 1)x + n = 0$$

の解になるので、これを解いて $p$ と $q$ を求めます。

```python
import math
a = 1
b = 1 + n - phi
c = n
D = b**2 - 4*a*c
assert D == math.isqrt(D) ** 2

p = (-b + math.isqrt(D))//2
q = (-b - math.isqrt(D))//2
assert p * q == n
```

あとは $m' = (m ^ e) ^ d \mod n$ なので、

$$(ed) \cdot \delta \equiv 1 \pmod{(p-1)(q-1)}$$

を満たす $\delta$ が求まれば

$$(m')^{\delta} \equiv m^{ed \cdot \delta} \equiv m \pmod n$$

でフラグを復元できます。

```python
d = pow(e, -1, phi)
delta = pow(e*d, -1, (p-1)*(q-1))
m_ = bytes_to_long(flag)
m = pow(m_, delta, n)
long_to_bytes(m)
```

### 逆元の存在について

普段は `e=65537` が素数であり、たまたま `(p-1)*(q-1)` が `e` の倍数になる確率はかなり低く、`pow(e, -1, (p-1)*(q-1))` が求まることが多いです。

今回は `pow(e*d, -1, (p-1)*(q-1))` を計算したいですが $g := \gcd(e\cdot d, (p-1)(q-1))$ が 1 でない可能性がそれなりにあります。

その場合は $m^{g}$ の $g$ 乗根を求める必要がありますが、今回は $g=1$ になるように作問者側で厳選してくれているそうで、簡単に $\delta$ を求めることができました。

## まとめ

$n$ の素因数と相関がある因数を含む $\phi$ があるとその関係式から $p$, $q$ が求まってしまう問題でした。

最近考えていた問題の解法に引っ張られて二次方程式を解いていましたが、改めて考えると

$$
\begin{align*}
(p-1)(q-1) &= (p+1)(q+1) - 2(p+q) \\
&= \phi - 2(\phi - n - 1)
\end{align*}
$$

なので､ $p$ と $q$ まで求めなくても $\delta$ を求めることができそうです。
