# PPPPParse

## 問題の概要

`JSON.parse` を 5 回適応した結果が `"42"` になるような文字列を入力する問題です。
ただし、`\\` が含まれる入力は Invalid 判定で弾かれます。

## 解法

JSON では文字列中で `\` を表す方法として、`\\` とエスケープする方法のほかに、`\u005c` と Unicode エスケープする方法もあります。

`"42"` を 5 回 `JSON.stringify` し、`\\` を `\u005c` に置換することで、条件を満たす入力を得ることができます。

## 解答に使用した入力

```
"\"\u005c\"\u005c\u005c\u005c\"\u005c\u005c\u005c\u005c\u005c\u005c\u005c\"42\u005c\u005c\u005c\u005c\u005c\u005c\u005c\"\u005c\u005c\u005c\"\u005c\"\""
```
