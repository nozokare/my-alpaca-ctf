# AAAAAAAAEEEEEEEESSSSSSSS

https://alpacahack.com/daily/challenges/aaaaaaaaeeeeeeeessssssss

## 問題の概要

AES-ECB モードで暗号化されたフラグを復号する問題です。

サーバーに接続するとフラグの各文字を 8 回繰り返した文字列が AES-ECB モードで暗号化されて送られてきます。

サーバーに任意の平文を送ると同じ鍵で暗号化した結果を返してくれます。

フラグに使用されている文字は [A-Za-z_{}] で、長さは 32 文字です。

## 解法

AES-ECB モードでは同じ平文のブロックは同じ暗号文のブロックになるため、ありうる平文のブロックのパターンを全て暗号化した結果を辞書にすれば、暗号文のブロックから平文のブロックを特定できます。

1 ブロックの長さが 16 バイトで、フラグの各文字が 8 回繰り返されているため、平文のブロックのパターンは 2 文字の組み合わせの全パターン 55 \* 55 通りになります。
