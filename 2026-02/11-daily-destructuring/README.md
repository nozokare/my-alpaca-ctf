# destructuring

https://alpacahack.com/daily/challenges/destructuring

## 問題の概要

JavaScript で `const { a: _0, b: _1 } = json;` のように分割代入したときに `_0`, `_1`, ..., に `0`, `1`, ... が代入されるような JSON 文字列を入力するとフラグが得られる問題です。

Stage 0 ～ Stage 4 までの 5 問があり、全問正解するとフラグが得られます。

例えば Stage 3 は次のような感じです。

```
const { a: { a: { a: { a: _0, b: _1 }, b: { a: _2, b: _3 }, c: [, _4, , _5, _6] }, b: [{ a: _7 }, [_8, _9, _10, , _11]] } } = json;
```

## 解法

分割代入の構造をそのまま反映した JSON オブジェクトを作ればよいです。

分割代入の文字列を次のように変換すれば目的の JSON 文字列が得られます。

- `a`, `b`, `c`, ... を `"a"`, `"b"`, `"c"`, ... に置換
- `_0`, `_1`, `_2`, ... を `0`, `1`, `2`, ... に置換
- `[, , _32]` のように配列内でスキップされている部分に `[null, null, 32]` のように `null` を挿入
- trailing comma を削除
