# super_simultaneous_equations

https://alpacahack.com/daily/challenges/super-simultaneous-equations

## 問題の概要

$n=2^{64}$ を法とする剰余類環 $Z/nZ$ 上の連立方程式を解くとフラグが得られる問題です。

コメントで案内がついている SageMath のプログラムが配布されており、$x_0$, ..., $x_4$ を変数とする 2 次多項式が 11 個与えられています。

```sage
polynomials = [
  5787757278604077443*x0^2 + 6778209859240936461*x0*x1 + ... + 13242867985814723662*x4^2 + 13201630723440635609,
  ...,
  x0 + 18428328970297319045
]
```

案内に従ってグレブナー基底を用いてこれらの多項式が全て 0 になるような $x_0$, ..., $x_4$ を求め、プログラムを実行するとフラグが得られます。

## 解法

次のようにグレブナー基底を求めると $x_0$ ～ $x_4$ の値が得られました。

```sage
G = ideal(polynomials).groebner_basis()
print(G)
# =>
# [x0 - 18415103412232571,
#  x1 - 8388008677569295199,
#  x2 - 7743220914302576998,
#  x3 - 6877970180684799795,
#  x4 - 8169375171471631741]
```

これらの値をプログラムに入力して実行するとフラグが得られます。

```sage
answer = {
    "x0": x0 - G[0],
    "x1": x1 - G[1],
    "x2": x2 - G[2],
    "x3": x3 - G[3],
    "x4": x4 - G[4],
}
def make_flag(answer):
    return b"".join([int(answer[i]).to_bytes(8, "big") for i in ["x0", "x1", "x2", "x3", "x4"]])

print(make_flag(answer))
```
